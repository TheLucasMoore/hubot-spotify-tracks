# hubot-spotify

grab an artist's top tracks

See [`src/spotify.coffee`](src/spotify.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-spotify --save`

Then add **hubot-spotify** to your `external-scripts.json`:

```json
[
  "hubot-spotify"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
